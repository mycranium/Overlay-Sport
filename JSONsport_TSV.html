Lower Third Text Preprocessor

Text Entry

Instructions

  • Paste the list of names into the textarea box below. This form takes tab-separated values, i.e. pasted directly from a spreadsheet.
  • If you want to clear the fields, click the Clear Fields button.
  • When you're ready to proceed, click the Process Text button.

Settings & Export

Instructions

  • Enter a 5-digit job number in the Global Settings: Job Number field (required). If the designer is someone other than the default, enter their name in the Designer field.
  • Select entries for editing by clicking the associated check box. You can use the options in the Select: header to help make selecting easier.
  • To change the default options, make selections from the dropdown menus. Click the Apply to Selected or Apply to All buttons depending on your needs.
  • You can edit the Version, Name, Title and Quote fields directly simply by clicking the text you wish to edit. Changes will be saved when you exit the field.
  • If you want to clear the whole page and start over, click the Reset Page button.
  • When you're ready to output the file, click the Make JSON button. This will download the JSON file and reset the page.
