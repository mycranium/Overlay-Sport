Instructions:

Build a color family here.

Color Family Name is required. It must contain only letters. Min 3 characters, max 15.

Color Names are optional. If used, they must start with a letter and contain only letters, numbers and spaces. Max 15 characters.

Hex Values are required. A leading hash character (#) is not required. Hex values must be vaild: six characters long (not including an optional leading hash character) containing only letters from a-f and numbers from 0-9. They are not case-sensitive.

Colors 1 through 3 are required. If you leave colors 4 or 5 empty, they will use the values from Color 3 and Color 1 respectively.
